Equipment Maintenance Request from Helpdesk Ticket

Maintenance Request from Helpdesk Support Ticket Odoo App

Only Compatible with Odoo Community Edition

This app allows your helpdesk team to create equipment maintenance requests from helpdesk support tickets.


Main Features:

  • Your support team can create an equipment maintenance request from the help desk ticket.
  • You can set a Responsible User, Maintenance Team, Employee and Equipment while creating a maintenance request from the ticket form.
  • You can see a Maintenance Request reference on the Helpdesk Ticket form as shown.
  • Helpdesk Support reference see on Maintenance Request form also available.
  • Record Rule for maintenance request for support user and manager to see all maintenance requests in the system has been added by app.
  • For more details please see below screenshots and watch the video.
