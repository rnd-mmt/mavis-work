Project Job Orders Request from Website

This app allows your customer / visitor of your website to create a job order or work order request from your website as shown.


Main Features:
  • Anyone (Customers / Visitors) can create a job order or work order request from your website by filling the request form as shown.
  • Allow customers to have a portal in My Account to view a list of job orders and print job orders.
  • Job orders / Work orders linked with Project so if customers fill Project code [optional] while submitting a request from the website then the created job order will be linked to specific Project in Odoo.
  • Instance email notification to customer by email when job order will be created.
  • Print Job Order PDF from backend as well by your internal team as sample shown.
  • Allow you to configure the job category in the backend and allow your customer to select from the request form.
  • Allow your team to send by email to your customer from the backend and attach a report of job order in the backend.
  • Integrated with Material Requisitions by Employees/Users so you can check material_purchase_requisitions app for more details about its features. Please click here: https://www.odoo.com/apps/modules/14.0/material_purchase_requisitions/.
  • For more details please watch the video and check below screenshots.
