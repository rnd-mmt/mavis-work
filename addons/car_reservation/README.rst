===============================
Vehicle/Car Reservation System
===============================

Vehicle & Car Reservation System Manages Vehicle Reservation and locations.

* This module brings nice features like:-

  - Provide autocomplete feature when you type location name for reservation purpose.
  - Reserve a Car (Vehicle) for Employee / Guest.
  - Google Map Graphical interface for Pick/Drop Location.
  - Occupancy of Car and Driver With Graphical Report.


=========
Features
=========

* Google Place Autocomplete Reservation Location

    - Provide an autocomplete feature when typing the name of a location (address).
    - Once you configure location, with the beauty of autocomplete feature address and longitude/latitude information will be rendered automatically.

* Geolocate for Reservation Location

    - Reservation Location have one button 'geolocate'
    - Click on the 'geolocate' button to update the latest longitude and latitude.

* Reserve a Car (Vehicle) for Employee / Guest

    - You can Book/Reserve car for Employee and Guest too with details like,
    - Employee / Guest to Pick/Drop (Fleet Manager),
    - Pickup and Drop Location,
    - Start and End timing of Reservation,
    - Reservation Approver (Fleet Manager),
    - Vehicle and Driver of it.

*  Graphical interface for Pick/Drop Location

    - Once you select Pickup and Drop Location you can easily navigate On Google Map graphical interface.

* Occupancy of Car and Driver With Graphical Report

    - To Generate the Report,
    - Go to Fleet --> Vehicles --> Car Reservations Menu --> Select Records.
